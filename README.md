## Description

### Interactive AMP-emails
I talked about AMP emails, a new generation of interactive emails with dynamic data.

Main points:
- The history of AMP-email creation. Examples with code.
- The main advantages of AMP emails. GET and POST requests. Working with forms. Calculations, conditional operators.
- Validation. Restrictions. Support by email clients. How to become an AMP-email sender.

I also prepared a live e-commerce AMP-email demo, which every conference participant could receive.


### Web3 Authentication in Solana Web Apps
I explained how to set up and integrate Web3 authentication into any web application, step by step.

Main points:
- The stages of Web3 authentication in a web application via a crypto wallet.
- How to determine which wallet the user is using.
- How to set up a connection via a wallet, what ready-made solutions are available.
- Signing the message with the user's private wallet key and then validating the signed transaction on the server side.
- Security, support, and expandability.

### Web3 Solana for JavaScript Developers
I explained what challenges a JavaScript developer faces when working with the Solana blockchain and the Web3.

Main points:
- How to connect to Solana blockhain via Web3.js library. Explain about the Connection class.
- Crypto Solana wallets. Public and private keys, ways to work with it.
- Transactions. Basic information, tips and tricks. 
- SPL tokens. How to create and send it via transactions.
- NFT. Types of NFTs. How to create and work with it.
